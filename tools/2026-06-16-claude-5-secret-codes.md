## Code 1: `199` (Numbered Prompt)

- **Model:** Opus 4.7
- **How:** Prefix your prompt with a number (e.g. `199`)
- **Example:** `199 how do I start implementing AI`
- **Effect:** Signals structured/deep prompting — Claude responds with a detailed, prioritized implementation plan

**Output observed:** 4-week speed-to-lead automation breakdown — Week 1: audit response times, Week 2: build AI responder (n8n/Make + LLM + calendar + CRM), Week 3: layer AI call summarization (Fireflies/Granola → CRM → follow-up emails), Week 4: measure lift.

## Code 2: `/GHOST` (Slash Command)

- **Model:** Sonnet 4.6
- **How:** Type `/ghost` + your request
- **Example:** `/ghost write me a short outreach email`
- **Effect:** Activates a writing mode that produces clean, human-sounding output

**Output observed:** Natural outreach email with subject line, personalized body, and a "Send via Gmail" button integration. Claude advised: *"fill in that second line with something real you noticed about them — that one detail separates a reply from a delete."*

**Discovered slash command menu:**
- `/campaign-planning` — content strategy, topics, editorial calendar
- `/content-strategy` — what content to create, topic clusters
- `/exploration-proposal-generator` — client proposals
- `/pitch-deck-generator` — pitch decks

## Code 3: `OODA` (Framework Prefix)

- **Model:** Sonnet 4.6
- **How:** Type `OODA` + your business context
- **Example:** `OODA write the best automation plan for my business`
- **Effect:** Forces Claude through the **Observe → Orient → Decide → Act** decision framework

**Output observed:**
- **Observe:** Mapped 3 businesses (Reprise AI, Sync2, Agents to Scale), identified key team members
- **Orient:** Found 4 bottlenecks — sales pipeline, content repurposing, sponsorship tracking, clinic onboarding
- **Decide:** Picked highest ROI: Automated Sales Follow-Up + Proposal Pipeline
- **Act:** Built full n8n workflow — Fireflies webhook → transcript → Claude classifies → auto-drafts email + proposal → Gmail draft → Airtable log. Build time: 2-3 hours.

- **How:** Type `SCAFFOLD` + what you want built

- **Step 3:** 5 core service lines — Voice Agents, Lead Follow-Up, Document Processing, AI Assistants, Reporting
- **Step 4:** Tools stack — n8n, Claude API, Notion, GoHighLevel
- **Step 5:** Sales system — 4-stage funnel (Lead Gen → Discovery → Proposal → Close)
- **Step 6:** Delivery system — 5-phase project flow
- **One-page summary:** Niche, offer, tech stack, Month 3 goal (2 clients, $10K), Month 12 goal ($30-50K/mo)
