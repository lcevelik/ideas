# 💡 Ideas & Thoughts

A place for project ideas, random thoughts, and brain dumps.

## Structure

- `projects/` — Concrete project ideas with rough specs
- `thoughts/` — General observations, half-baked ideas, musings
- `research/` — Interesting things to look into later

## Usage

Just drop `.md` files in here. No rules, no structure required — just get the ideas out.
